# XHS Account Bootstrapping

## Overview

面向“小红书起号/养号”场景的可执行流程。  
目标是让新号在不违规前提下，尽快建立账号标签、内容稳定性与首批正向互动。

- 标题模式高频：`结果承诺型(9/20)`、`周期打卡型(4/20)`、`教程方法型(4/20)`、`避坑警示型(3/20)`
- 内容侧高频主题：`互动行为`、`更新节奏`、`定位明确`、`避坑风控`
- 实操启示：起号不是“单条爆文技巧”，而是“连续行为 + 账号一致性”

## 4-Layer 起号框架

### Layer 1：账号层（Who）

- 人设一句话：`我是谁 + 为谁解决什么问题`
- 赛道收敛：前 14 天尽量不跨垂类乱跳

### Layer 2：内容层（What）

- 建立 3 类内容池：
  - `A池`：高确定性干货（教程/清单/模板）
  - `B池`：进展复盘（DayX、踩坑、调整）
  - `C池`：轻互动内容（提问、投票、对比）

建议占比：`A:B:C = 5:3:2`

### Layer 3：分发层（How）

- 标题优先写“结果+对象”：如“普通人/新手/大学生 + 可达结果”
- 首屏前 3 行交代痛点与收益
- 首发后 30 分钟内完成第一轮互动

### Layer 4：反馈层（Improve）

- 周维度复盘，不日内过度焦虑
- 每周至少做 1 次 AB 测试（标题或封面）
- 只放大前 20% 表现选题，果断淘汰低效题材

## When to Use

- 新账号刚创建，准备开始发内容
- 老账号长期低曝光，准备重建发布节奏
- 需要把“养号建议”落地为可检查的动作清单

## Input Contract

执行前先明确：

- 赛道：如美妆/宠物/学习/职场/家居
- 人设：你是谁、给谁看、解决什么问题
- 内容形态：图文 or 视频
- 可投入频次：每周可发布几条

## 0-28 天起号流程

### Day 0（建档）

1. 完成账号基础资料：头像、昵称、简介、背景图。
2. 用一句话固定账号定位：`我是谁 + 给谁看 + 提供什么价值`。
3. 预先准备 10-15 个同赛道选题，避免断更。

### Day 1-7（标签建立期）

1. 连续发布 4-7 条同赛道内容，不跨赛道乱跳。
2. 每条内容结构固定：痛点开头 → 过程/方法 → 结果/总结。
3. 每条发布后 30 分钟内做首轮互动（回复评论、回访同赛道）。
4. 严禁刷量互赞、批量关注取关、搬运洗稿。

### Day 8-14（稳定放量期）

1. 复盘前 7 天数据：点赞率、收藏率、评论率、完读/完播。
2. 保留前 20% 表现选题，淘汰低表现题材。
3. 强化“可复用栏目”：如 `DayX 进展`、`每周一次总结`。
4. 发布频次保持稳定（宁可少发，也不要忽快忽慢）。

### Day 15-28（优化迭代期）

1. 每周至少 1 次 AB 测试（标题/封面/开头前 3 秒）。
2. 增加“真人互动感”：评论区追问、更新进展、失败复盘。
3. 对高互动内容做二次创作（延伸篇/FAQ/反例篇）。
4. 建立素材库与复盘库，进入可持续发布状态。

## 热帖同款结构模板

### 模板 A：周期承诺型（高点击）

- 标题：`X天起号/打卡 + 结果预期`
- 正文：`目标 -> 当天动作 -> 数据反馈 -> 次日计划`

### 模板 B：避坑警示型（高收藏）

- 标题：`X个伤号行为/千万不要做`
- 正文：`错误动作 -> 为什么伤号 -> 替代做法`

### 模板 C：新手进展型（高评论）

- 标题：`DayN 进展 + 真实数据`
- 正文：`今天做了什么 -> 遇到的问题 -> 明天怎么改`

### 模板 D：身份结果型（高点击）

- 标题：`普通人/新手/大学生 + 结果数字`
- 正文：`起点条件 -> 关键动作 -> 数据变化 -> 可复用建议`

## 评论与互动规则

1. 优先回复“有信息量”的评论，而不是只回表情。
2. 互动回复要带细节，不要机械“谢谢支持”。
3. 主动抛 1 个问题引导讨论，提升评论深度。

## 风险红线（必须遵守）

- 不刷赞刷评刷粉
- 不搬运/洗稿/盗图
- 不跨平台违规引流
- 不频繁删除重发同内容
- 不在短时间内做异常高频操作

## Weekly Review Checklist

- [ ] 本周发布是否保持稳定频次？
- [ ] 账号定位是否保持一致？
- [ ] 前 3 秒/首屏表达是否更清晰？
- [ ] 评论区是否形成有效互动？
- [ ] 是否出现潜在违规动作？
- [ ] 下周保留与淘汰选题是否明确？
