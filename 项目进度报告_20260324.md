### 遇到问题 ⚠️

1. **浏览器自动化阻塞**
   - 现象：同步进程卡在"创建新专辑"步骤
   - 原因：小红书网页版需要有效的登录会话
   - 影响：无法完全自动化执行

2. **直接 API 调用失败**
   - 现象：API 返回 404 或解析错误
   - 原因：小红书 API 需要浏览器上下文的特殊 headers
   - 影响：无法绕过浏览器直接调用
