## 🔍 CSP 绕过方案验证

### 方案 1: fetch API
```javascript
fetch(url, {method: 'POST', ...})
```
**结果**: ❌ Failed to fetch  
**原因**: CSP 直接阻止

---

### 方案 2: XMLHttpRequest
```javascript
xhr.open('POST', url);
xhr.send(data);
```
**结果**: ❌ status 0  
**原因**: CSP 阻止网络请求

---

### 方案 3: 注入全局函数
```javascript
window.callXiaohongshuAPI = function() {...}
```
**结果**: ❌ 导航后清除  
**原因**: 页面重新加载后注入的函数丢失

---

### 方案 4: AddScriptToEvaluateOnNewDocument
```javascript
page.AddScriptToEvaluateOnNewDocument(script)
```
**结果**: ❌ 仍然被 CSP 阻止  
**原因**: 小红书 CSP 策略严格

---

### 方案 5: CDP Runtime.evaluate
```javascript
page.Eval(`(async () => { fetch(...) })()`)
```
**结果**: ❌ JS 语法错误  
**原因**: 字符串拼接导致语法问题
