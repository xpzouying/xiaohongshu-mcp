# 技术方案：收藏夹自动分类功能

## 功能需求

### 核心功能
1. **获取收藏列表** - 获取用户所有收藏的笔记
2. **自动分类** - AI 分析笔记内容，自动打标签分类
3. **筛选查询** - 根据标签/提示词筛选收藏笔记
4. **收藏夹管理** - 创建/管理收藏夹，移动笔记到指定分类

### 用户场景
```
场景 1: 查看我的收藏
→ 调用 GET /api/v1/favorites/list
→ 返回收藏笔记列表（含标题、封面、收藏时间等）

场景 2: 自动分类收藏
→ 调用 POST /api/v1/favorites/classify
→ AI 分析所有收藏，生成分类标签
→ 返回分类结果

场景 3: 筛选收藏笔记
→ 调用 POST /api/v1/favorites/filter
→ 传入标签/关键词："健身食谱"
→ 返回匹配的收藏笔记

场景 4: 整理收藏夹
→ 调用 POST /api/v1/favorites/organize
→ 将筛选出的笔记移动到指定收藏夹
```

### Phase 1: 获取收藏列表

#### 1.1 新增文件 `xiaohongshu/favorite_list.go`

```go
package xiaohongshu

import (
    "context"
    "encoding/json"
    "time"
    "github.com/go-rod/rod"
)

// FavoriteItem 收藏笔记项
type FavoriteItem struct {
    FeedID      string    `json:"feed_id"`
    XsecToken   string    `json:"xsec_token"`
    Title       string    `json:"title"`
    Desc        string    `json:"desc"`
    CoverURL    string    `json:"cover_url"`
    UserNickname string   `json:"user_nickname"`
    UserID      string    `json:"user_id"`
    LikedCount  int       `json:"liked_count"`
    CollectedCount int    `json:"collected_count"`
    CollectTime time.Time `json:"collect_time"`
}

// FavoriteListResponse 收藏列表响应
type FavoriteListResponse struct {
    Items []FavoriteItem `json:"items"`
    Count int            `json:"count"`
    HasMore bool         `json:"has_more"`
    Cursor string        `json:"cursor"`
}

// FavoriteListAction 收藏列表操作
type FavoriteListAction struct {
    page *rod.Page
}

func NewFavoriteListAction(page *rod.Page) *FavoriteListAction {
    return &FavoriteListAction{page: page}
}

// GetFavoriteList 获取收藏列表
func (a *FavoriteListAction) GetFavoriteList(ctx context.Context, cursor string, pageSize int) (*FavoriteListResponse, error) {
    page := a.page.Context(ctx).Timeout(120 * time.Second)
    
    // 1. 导航到个人主页
    profileURL := "https://www.xiaohongshu.com/user/profile/me"
    page.MustNavigate(profileURL)
    page.MustWaitDOMStable()
    time.Sleep(2 * time.Second)
    
    // 2. 点击"收藏"标签页
    collectTab := page.MustElementR(".tabs .tab", "收藏")
    collectTab.MustClick()
    time.Sleep(2 * time.Second)
    
    // 3. 滚动加载更多内容
    if cursor != "" {
        // 实现滚动加载逻辑
        a.scrollToLoadMore(page, cursor)
    }
    
    // 4. 解析收藏列表数据
    items, err := a.parseFavoriteItems(page)
    if err != nil {
        return nil, err
    }
    
    return &FavoriteListResponse{
        Items: items,
        Count: len(items),
        HasMore: len(items) >= pageSize,
        Cursor: generateCursor(items),
    }, nil
}

// parseFavoriteItems 解析收藏笔记列表
func (a *FavoriteListAction) parseFavoriteItems(page *rod.Page) ([]FavoriteItem, error) {
    // 从 __INITIAL_STATE__ 或 DOM 中解析数据
    script := `() => {
        if (window.__INITIAL_STATE__ && 
            window.__INITIAL_STATE__.user && 
            window.__INITIAL_STATE__.user.collect) {
            return JSON.stringify(window.__INITIAL_STATE__.user.collect);
        }
        return "[]";
    }`
    
    result := page.MustEval(script).String()
    if result == "[]" {
        return []FavoriteItem{}, nil
    }
    
    var collectData struct {
        Items []struct {
            NoteID  string `json:"noteId"`
            Note    struct {
                Title  string `json:"title"`
                Desc   string `json:"desc"`
                Images []struct {
                    URLDefault string `json:"urlDefault"`
                } `json:"images"`
                User struct {
                    Nickname string `json:"nickname"`
                    UserID   string `json:"userId"`
                } `json:"user"`
                InteractInfo struct {
                    LikedCount  int `json:"likedCount"`
                    CollectedCount int `json:"collectedCount"`
                } `json:"interactInfo"`
            } `json:"note"`
            CollectTime int64 `json:"collectTime"`
        } `json:"items"`
    }
    
    if err := json.Unmarshal([]byte(result), &collectData); err != nil {
        return nil, err
    }
    
    // 转换为 FavoriteItem
    items := make([]FavoriteItem, 0, len(collectData.Items))
    for _, item := range collectData.Items {
        coverURL := ""
        if len(item.Note.Images) > 0 {
            coverURL = item.Note.Images[0].URLDefault
        }
        
        items = append(items, FavoriteItem{
            FeedID: item.NoteID,
            Title:  item.Note.Title,
            Desc:   item.Note.Desc,
            CoverURL: coverURL,
            UserNickname: item.Note.User.Nickname,
            UserID: item.Note.User.UserID,
            LikedCount: item.Note.InteractInfo.LikedCount,
            CollectedCount: item.Note.InteractInfo.CollectedCount,
            CollectTime: time.Unix(item.CollectTime/1000, 0),
        })
    }
    
    return items, nil
}

func (a *FavoriteListAction) scrollToLoadMore(page *rod.Page, cursor string) {
    // 实现滚动加载逻辑
    page.MustEval(`() => window.scrollTo(0, document.body.scrollHeight)`)
    time.Sleep(1 * time.Second)
}

func generateCursor(items []FavoriteItem) string {
    if len(items) == 0 {
        return ""
    }
    // 使用最后一项的时间戳作为 cursor
    return items[len(items)-1].CollectTime.Format(time.RFC3339)
}
```

### Phase 2: AI 自动分类

#### 2.1 新增文件 `xiaohongshu/favorite_classifier.go`

```go
package xiaohongshu

import (
    "context"
    "fmt"
)

// ClassificationResult 分类结果
type ClassificationResult struct {
    FeedID    string   `json:"feed_id"`
    Title     string   `json:"title"`
    Category  string   `json:"category"`
    Tags      []string `json:"tags"`
    Confidence float64 `json:"confidence"`
}

// FavoriteClassifier 收藏夹分类器
type FavoriteClassifier struct {
    // 可以集成 AI API 客户端，如 OpenAI、Claude 等
    apiClient *AIClient
}

// ClassifyFavorites 对收藏笔记进行分类
func (c *FavoriteClassifier) ClassifyFavorites(ctx context.Context, items []FavoriteItem) ([]ClassificationResult, error) {
    results := make([]ClassificationResult, 0, len(items))
    
    // 批量处理，每 10 个笔记调用一次 AI
    batchSize := 10
    for i := 0; i < len(items); i += batchSize {
        end := i + batchSize
        if end > len(items) {
            end = len(items)
        }
        
        batch := items[i:end]
        batchResults, err := c.classifyBatch(ctx, batch)
        if err != nil {
            return nil, err
        }
        
        results = append(results, batchResults...)
    }
    
    return results, nil
}

func (c *FavoriteClassifier) classifyBatch(ctx context.Context, items []FavoriteItem) ([]ClassificationResult, error) {
    // 构建提示词
    prompt := c.buildClassificationPrompt(items)
    
    // 调用 AI API
    response, err := c.apiClient.Call(ctx, prompt)
    if err != nil {
        return nil, err
    }
    
    // 解析 AI 返回的分类结果
    return c.parseClassificationResponse(response, items)
}

func (c *FavoriteClassifier) buildClassificationPrompt(items []FavoriteItem) string {
    // 构建分类提示词
    return fmt.Sprintf(`请分析以下小红书笔记，为每个笔记分类：
分类类别：美食、旅行、学习、穿搭、美妆、健身、家居、科技、娱乐、其他

笔记列表：
%v

请以 JSON 格式返回分类结果，包含：feed_id, category, tags, confidence`, items)
}
```

## 风险和挑战

1. **小红书反爬虫**: 频繁请求可能触发风控
   - 解决：添加请求延迟、使用真实用户行为模拟

2. **UI 结构变化**: 网页版更新可能导致选择器失效
   - 解决：使用更稳定的选择器，添加错误重试

3. **AI 分类准确性**: 分类可能不准确
   - 解决：提供人工修正接口，支持用户自定义分类

4. **收藏夹管理 API**: 小红书可能不开放此功能
   - 解决：使用浏览器自动化模拟用户操作
